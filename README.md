Zastosowania-Informatyki - Publikowanie własnej biblioteki Maven z GitHub Packages
---

## Wprowadzenie
Czyli o tym jak w praktyce opublikować własną bibliotekę w Javie do repozytorium Maven na GitHub Packages. 
Jest to realne zadanie spotykane w pracy programisty, gdy tworzymy kod współdzielony w zespole lub własne narzędzia. 
Pokazany zostanie proces od stworzenia prostej biblioteki, przez konfigurację Mavena, po zautomatyzowane publikowanie w GitHub Actions.

---

## Cel 
- stworzenie prostej biblioteki Java
- konfiguracja publikacji w `pom.xml`
- publikacja pakietu na GitHub Packages
- automatyzacja przez GitHub Actions
- użycie biblioteki w innym projekcie

**Celem** jest pokazanie pełnego, praktycznego przepływu CI/CD związanego z publikacją artefaktów Maven.
